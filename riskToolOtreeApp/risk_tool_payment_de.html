Nun möchten wir Ihnen mit Hilfe eines interaktiven Programms die Risiken bei verschiedenen Anlagemöglichkeiten veranschaulichen, bevor Sie anschließend eine Anlageentscheidung treffen.

Stellen Sie sich dazu vor, dass Sie {{ C.budget_string }} Euro über einen Zeitraum von {{ C.Anlagehorizont }} Jahren anlegen möchten. Sie können das Geld in zwei verschiedene Anlageformen investieren: in risikobehaftete Aktien oder in risikofreie Anleihen. Sie können das Geld in einer der beiden Anlageformen investieren oder in beide Anlageformen gleichzeitig. Sie können auch entscheiden, wie viel Geld Sie in jede Anlageform investieren möchten.

Wichtig:

Jede/r zehnte Teilnehmende dieser Studie wird den tatsächlichen Wert des simulierten Portfolios nach einem Anlagezeitraum von {{ C.Anlagehorizont }} Jahren in Cent ausgezahlt bekommen. Ein Endwert des simulierten Portfolios von z.B. {{ C.budget_string }} Euro enspricht also einer Auszahlung von {{ Auszahlung }} Euro. Die Simulation basiert auf historischen Erträgen (Renditen) breit gestreuter Anlagen. Die Auszahlung erfolgt innerhalb eines Monats nach Ihrer Teilnahme.
